jQuery/Zepto 页面弹窗插件

weui 
<script type="text/javascript" src="jquery.js"></script>
<script type="text/javascript" src="alert.js"></script>
<script>
	$.alert('弹出了一个alert对话框')
	$.confirm('弹出了一个confirm对话框',function(e){
		$.alert('你点击了'+(e?'确定':'取消'))
	})
	$.tips('显示了一条tips')
	$.load('显示了一条load信息')
</script>
方法 参数 参数说明 方法说明
$.alert string 必须，弹窗消息 弹出一个只有确定按钮的对话框
function 可选，回调函数
$.confirm string 必须，弹窗消息 弹出一个有确定和取消两个按钮的对话框
function 可选，回调函数
$.tips string 必须，消息内容 弹出一个会自动消失的tips提示框
int 可选，自动关闭的时间，仅PC浏览器有效
$.load string 可选，消息内容 弹出一个没有任何操作按钮的对话框，需要手动关闭
$.loaded 无 关闭load对话框
